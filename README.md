# Abra PowerShell
Inicie el comando:

```
iwr -useb http://winutil.migzam.com | iex
```
o el siguiente comando para ejecutar como Administrador. 
```
Start-Process powershell.exe -Verb RunAs -ArgumentList "iwr -useb http://winutil.migzam.com | iex"
```
